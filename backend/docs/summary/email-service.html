E-mailová služba

Asynchronní rozesílání transakčních e-mailů pro registraci členů a nastavení hesel s šablonami a robustním ošetřením chyb.

Event-driven architektura

E-maily se odesílají asynchronně po úspěšném commitu transakce. Selhání SMTP serveru neblokuje registraci člena.

Hlavní funkce

Uvítací e-mail

Automatický e-mail po registraci s aktivačním odkazem a registračním číslem člena.

Aktivační tokeny

Bezpečné, časově omezené tokeny (72 hodin) pro aktivaci účtu, jednorázové použití.

E-mailové šablony

HTML i plain-text verze se substitucí proměnných (jméno, registrační číslo, aktivační URL).

Event-driven odesílání

Transakční event listener odesílá e-maily až po commitu transakce do databáze.

Graceful failure

Selhání e-mailu nezpůsobí rollback registrace. Chyby se logují bez PII dat.

Konfigurace

SMTP nastavení, from adresa, platnost tokenů a base URL pro aktivační linky.

Typy e-mailů
  • Uvítací e-mail: Pro dospělé členy na jejich e-mail, pro nezletilé na e-mail zákonného zástupce
  • Žádný e-mail: Pokud člen nemá vyplněný e-mail (varování logováno)
  • Password setup e-mail: S linkem pro nastavení počátečního hesla
Aktivační token

Specifikace tokenu

Formát: UUID v4 (36 znaků)
Platnost: 72 hodin (konfigurovatelné)
Použití: Jednorázové (po aktivaci se invaliduje)
Storage: Cryptographically hashovaný před uložením do DB

E-mailové šablony
  • HTML šablona: Pro e-mailové klienty s podporou HTML
  • Plain-text šablona: Pro textové e-mailové klienty
  • Multi-part: Oba formáty v jednom e-mailu (multipart/alternative)
  • Proměnné: Jméno, příjmení, registrační číslo, aktivační URL, název klubu
  • Externalizace: Šablony nejsou hardcoded, ale uloženy v konfiguraci
Ošetření chyb

Graceful degradation

SMTP selhání: E-mail se neodešle, ale registrace proběhne v pořádku. Chyba se loguje.
Nevalidní e-mail: Logováno, e-mail se neodesílá.
Chyba šablony: Fallback na plain-text verzi, chyba se loguje.
Logování bez PII: Pouze registrační číslo, žádné osobní údaje.

Konfigurovatelné parametry
  • SMTP host, port, credentials - Připojení k e-mailovému serveru
  • From e-mail adresa - Odesílatel e-mailů
  • Platnost aktivačního tokenu - Výchozí 72 hodin
  • Base URL - Základní URL pro aktivační linky
