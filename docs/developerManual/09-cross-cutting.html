Cross-cutting concerns

Sdílená infrastruktura napříč moduly: bootstrap dat při startu, MDC logování s correlation ID a per-key rate limiting. Spring Modulith eventy a metriky popisuje kap. 08; výjimky a Encryption najdete v kap. 07 resp. kap. 03.

Bootstrap dat

Plugin SPI pro inicializaci dat při startu aplikace. Každý modul, který potřebuje seedovat výchozí data, implementuje BootstrapDataInitializer a registruje jako @Component. BootstrapDataLoader (jeden ApplicationRunner) projde všechny implementace a v případě potřeby je spustí. 

public interface BootstrapDataInitializer {
    boolean requiresBootstrap();    // true = data ještě nejsou
    void bootstrapData();
}
@Component
class MembersDataBootstrap implements BootstrapDataInitializer {
    public boolean requiresBootstrap() { return memberRepo.existsAny() == false; }
    public void bootstrapData() { /* vytvoř seed members */ }
}
Existující implementace: MembersDataBootstrap, UsersDataBootstrap, OidcRegisteredClientsBootstrap (OAuth2 klienti), TrainingGroupDataBootstrap. OAuth2 klient klabis-web-local se zavádí jen při profilu local-dev — pro povolení refresh tokenů z localhost frontendu.

MDC logging — RequestLoggingFilter

Filter v common.logging, který obohatí MDC o context aktuálního HTTP requestu. Aktivní jen pro /api/** endpointy.

MDC klíčHodnota
requestContext"GET /members/123" (HTTP method + cesta bez /api)
userIdUUID přihlášeného uživatele (pokud autentizován)
correlationIdZ X-Correlation-ID nebo X-Request-ID headeru, jinak vygenerované UUID

Tyto klíče lze použít v logback-spring.xml patternech (%X{requestContext}). Pomocné konvertery HighlightLevelConverter a ThreadOrRequestConverter jsou v stejném package.

PerKeyRateLimiter

Cache-driven rate limiter — každý klíč (např. registrační číslo, IP) má vlastní counter s periodickým resetem. Vyžaduje nakonfigurovanou Spring Cache s názvem rateLimitCache. 

public class PerKeyRateLimiter {
    public PerKeyRateLimiter(int limit, Duration duration, CacheManager cacheManager);

    public void checkLimit(String key);   // throws RateLimitExceededException
}
// Konfigurace beanu
@Bean
PerKeyRateLimiter passwordSetupRateLimiter(CacheManager cm) {
    return new PerKeyRateLimiter(5, Duration.ofMinutes(15), cm);
}

// Použití
rateLimiter.checkLimit(registrationNumber);  // throws if exceeded
RateLimitExceededException se mapuje v MvcExceptionHandler na HTTP 429.
